# 🔊 Meme Error Audio

> Because your code errors deserve a reaction.

A VS Code extension that plays viral Indian meme sounds whenever errors are detected in your code. No more silently staring at red squiggles — now they come with sound effects! 😂

---

## 🎵 Default Sounds

- **Faah** — that iconic exaggerated exhale
- **Kya Cheda Bhos\*\*** — for when your code is just unnecessarily annoying

Sounds are picked **randomly** by default, or you can choose your favorite!

---

## ✨ Features

- 🐛 Plays meme audio automatically when code errors are detected
- 🎵 Choose between default sounds or set your own custom `.mp3`
- ⚡ Works with **any programming language** in VS Code
- 🔇 Debounced — won't spam sounds while you're actively typing

---

## ⚙️ Settings

| Setting | Default | Description |
|---|---|---|
| `memeErrorAudio.defaultSound` | `random` | Choose `faah`, `kya-cheda`, or `random` |
| `memeErrorAudio.customSoundPath` | `` | Full path to your own `.mp3` file |

### How to set a custom sound (Windows)
1. Find your `.mp3` file in File Explorer
2. Hold `Shift` + Right click → **"Copy as path"**
3. Open VS Code Settings (`Ctrl+,`)
4. Search **"Meme Error Audio"**
5. Paste the path into **Custom Sound Path**

## 🚀 Installation

Install from the [VS Code Marketplace](#) *(link coming soon)*

Or clone and run locally:
```bash
git clone https://github.com/piyushkumar-git/meme-error-audio.git
cd meme-error-audio
npm install
```
Then press `F5` in VS Code to run!
